Урок 03

Собираем еженедельный отчёт

Соединим регулярные данные и ИИ-резюме, чтобы быстрее видеть главное за неделю.

Проблемная ситуация

К концу недели данные разбросаны по таблицам, а итог приходится собирать вручную.

Прогноз решения

Сценарий может собрать показатели по расписанию и передать их ИИ для короткого резюме.

Короткое объяснение

Для надёжного отчёта отделяйте факты от интерпретаций: сначала передавайте цифры, затем просите выделить тенденции.

Частично собранный пример

Запрос к ИИНа основе показателей составь резюме из 3 пунктов. Не добавляй данных, которых нет во входных данных. Отдельно отметь изменения больше 15%.

Самостоятельная сборка

  1. Определите 3–5 метрик отчёта.
  2. Настройте еженедельный запуск.
  3. Отправьте черновик отчёта в нужный канал.

Диагностика ошибок

Если цифры не совпадают, сначала проверьте источник и формат дат. Если выводы слишком смелые — попросите ИИ ссылаться на конкретную метрику.

Перенос навыка

Подход работает для отчётов по продажам, контенту, обучению и поддержке.
