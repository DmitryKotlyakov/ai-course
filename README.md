# ИИ в автоматизации

Учебный сайт курса по использованию ИИ в автоматизации рабочих процессов.

## Запуск локально

```powershell
npm install
npm run dev
```

Откройте адрес, который покажет Vite (по умолчанию `http://localhost:5173/`). Уроки доступны по адресам `/lesson-1`, `/lesson-2` и `/lesson-3`.

## Структура

- `index.html` — главная страница и программа курса.
- `lessons/` — HTML-содержимое самостоятельных уроков.
- `vite.config.js` — настройки Vite и чистые маршруты уроков для локальной разработки.
- `assets/css/` — стили интерфейса и уроков.
- `assets/js/` — небольшие интерактивные элементы.
